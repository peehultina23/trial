Practical 9: Introduction to DApps

Steps:
1. Add custom network in MetaMask using Ganache RPC URL.
2. Import account from Ganache to MetaMask using private key.
3. Initialize Truffle project and add Add.sol contract.
4. Add migration script 1_deploy.js.
5. Configure truffle-config.js for network and compiler version.
6. Compile and deploy contract using Truffle.
